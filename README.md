## Installation
1. `apt install git -y`
2. `cd $HOME`
3. `git clone https://github.com/Mrp1r4t3/FBDO`
4. `cd Bl1x0rz`
5. `bash install.sh`

## Uninstallation
1. `cd $HOME`
2. `cd Bl1x0rz`
3. `bash uninstall.sh`
